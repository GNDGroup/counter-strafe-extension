# CounterStrafe — расширение для Chrome

Показывает аналитику прямо на странице матча в FACEIT: статы всех 10 игроков, рекомендации по картам, кто пикать, кто банить.

## Установка

1. Скачай zip из [Releases](https://github.com/GNDGroup/counter-strafe-extension/releases) и распакуй
2. Открой Chrome, перейди на `chrome://extensions/`
3. Включи **Режим разработчика** (переключатель справа вверху)
4. Нажми **«Загрузить распакованное»**
5. Выбери папку `chrome-extension`

Всё, расширение работает.

## Как пользоваться

1. Зайди на [counterstrafe.pro](https://counterstrafe.pro) и авторизуйся через FACEIT
2. Открой любую комнату матча на FACEIT
3. Снизу появится панель со статами игроков и картами

## Что показывает

- ELO и уровень каждого игрока
- K/D, винрейт
- Какие карты пикать, какие банить — с учётом стата обеих команд
- Подсветка карт прямо на странице во время вето
