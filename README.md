# CounterStrafe Analytics — Chrome Extension

Injects a pre-match analytics panel directly into FACEIT match room pages, showing ELO, K/D, win rate, and skill level for all 10 players.

## Installation (Developer Mode)

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right corner)
3. Click **"Load unpacked"**
4. Select the `chrome-extension/` folder from this repo
5. The extension is now active

## Usage

1. Log in to [counterstrafe.pro](https://counterstrafe.pro) (required for analytics)
2. Open any FACEIT match room: `faceit.com/*/room/<matchID>`
3. A **CounterStrafe Analytics** panel appears on the right side of the page
4. Click the `←` button to collapse/expand the panel

## Features

- **Auto-inject** — appears automatically on FACEIT match room pages
- **Player stats** — ELO, skill level, K/D ratio, win rate for all 10 players
- **CT/T split** — teams color-coded by side (blue = CT, amber = T)
- **Collapsible** — panel state persists across page navigations
- **SPA-aware** — handles FACEIT's React Router navigation without reloads
- **Auth** — reads your `cs_token` cookie from counterstrafe.pro for API calls

## Requirements

- Must be logged into [counterstrafe.pro](https://counterstrafe.pro) to fetch analytics
- Match must have been played and processed by CounterStrafe (demos ready ~15 min after match)

## Development

No build step required. Edit files directly and reload the extension in `chrome://extensions/`.

### File structure

```
chrome-extension/
├── manifest.json      — Extension manifest (Manifest V3)
├── content.js         — Injected into FACEIT match room pages
├── content.css        — Panel styles
├── background.js      — Service worker: handles API calls + cookie auth
├── popup/
│   ├── popup.html     — Extension popup UI
│   ├── popup.js       — Popup logic
│   └── popup.css      — Popup styles
├── icons/
│   ├── icon16.png     — Generated by generate-icons.js
│   ├── icon48.png
│   ├── icon128.png
│   └── generate-icons.js  — Icon generator (run with node)
└── README.md
```

### Permissions used

- `storage` — panel collapsed/expanded state
- `activeTab` — check if current tab is on FACEIT
- `cookies` — read `cs_token` from counterstrafe.pro for auth
- Host: `faceit.com` — inject panel
- Host: `counterstrafe.pro` — API calls
